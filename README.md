# VoteGuard App

A minimal Vite + React setup to run the provided VoteGuard UI.

## Requirements

- Node.js 18+

## Install & Run

```bash
cd /Users/mouniksai/Desktop/cloud
npm install
npm run dev
```

Then open the shown local URL (usually http://localhost:5173).

## Build & Preview

```bash
npm run build
npm run preview
```
